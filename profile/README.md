## What is ModelSense?
 
ModelSense is an AI powered model recommendation system that analyzes your software project and identifies the optimal combination of AI models for cost-efficiency, performance, and reliability.
 
Developers and teams face hundreds of model choices across OpenAI, Anthropic, Google, Mistral, and more, each with different pricing, strengths, and tradeoffs. Choosing wrong means overpaying, underperforming, or rebuilding.
 
Describe your project and ModelSense returns a tailored model recommendation with cost estimates, provider comparisons, and clear reasoning behind every pick.
 
## How it works
 
1. **Describe your project**: what you're building, its AI tasks, expected scale, and budget
2. **Get a tailored recommendation**: ModelSense decomposes your project into distinct AI tasks and maps each to the best-fit model
3. **See the full picture**: cost estimates, provider comparisons, and the reasoning behind every pick
 
## Features
 
### Analysis
- Decomposes your project into distinct AI task types
- Recommends the best fit model per task, with reasoning
- Budget / Balanced / Premium model tiers for every task
- Model capability badges: context window, multimodal support, open source availability, data residency
### Cost breakdown
- Per-1K token pricing for every recommended model
- Savings estimate vs using GPT-4o for everything
- API vs subscription breakeven guidance
- Cost estimates at light, medium, and heavy usage scale
- Custom scale input for a personalised estimate
- Per-query and per-MAU cost framing
### Customisation
- Covers all major providers — OpenAI, Anthropic, Google, Mistral, DeepSeek
- Restrict recommendations to specific vendors
### Projects
- Save and revisit past project analyses
- Revise and re-run analyses as your project evolves
